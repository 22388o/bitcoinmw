Macro bmw_util::stack

source · 
macro_rules! stack {
    ( $size:expr, $default:expr ) => { ... };
}
Expand description

The crate::stack macro creates a crate::Stack implementation with the specified parameters. Note that like crate::Queue, this is a bounded stack. The stack is returned as an impl Stack<T>.

§Input Parameters

  • size (usize) (required) - the size of the underlying array
  • default (required) - a reference to the value to initialize the array with for the stack, these values are never used, but a default is needed to initialize the underlying array.

§Return

Returns Ok(impl Stack<T>) on success and a [bmw_err::Error] on failure.

§Errors

  • [bmw_err::ErrKind::IllegalArgument] - if the size is 0.

§Examples

 use bmw_err::*;
 use bmw_log::*;
 use bmw_util::*;

 fn main() -> Result<(), Error> {
         let mut stack = stack!(10, &0)?;

         for i in 0..10 {
                 stack.push(i)?;
         }

         for i in (0..10).rev() {
                 let v = stack.pop().unwrap();
                 assert_eq!(v, &i);
         }

         Ok(())
 }
