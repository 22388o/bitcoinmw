Macro bmw_log::log_all

macro_rules! log_all {
    ($level:expr) => { ... };
    ($level:expr, $a:expr) => { ... };
    ($level:expr, $a:expr, $($b:tt)*) => { ... };
}
Expand description

This macro is called by the fatal_all, error_all, warn_all, info_all, debug_all and trace_all macros to process logging. Generally those macros should be used in favor of calling this one directly.
